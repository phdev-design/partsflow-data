# partsflow-data

Static assets (diagram images, PDF catalogues, structured JSON) for the
PartsFlow application, served via [jsDelivr](https://www.jsdelivr.com/).

Do not put application code here — this repo only holds data so the main
`partsflow` repo and server stay lean.

## CDN base

```
https://cdn.jsdelivr.net/gh/phdev-design/partsflow-data@main/
```

## Layout

```
rifter-k9/
  images/*        # exploded-view diagrams
  pdf/*.pdf       # catalogue PDFs
  data.json       # structured multilingual parts data
traveller-k0/
  images/* pdf/* data.json
208/
  images/* pdf/* data.json
2008-p24e/
  images/* pdf/* data.json
sprinter-w907/
  images/*        # Mercedes Sprinter W907 (PartSouq mirror)
```

Regenerated by `scripts/build_data_repo.sh` in the main repo.
